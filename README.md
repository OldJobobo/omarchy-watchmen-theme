# Watchmen Omarchy Theme

An Omarchy theme based on the original *Watchmen* graphic novel series, built from ink-dark panels, paper-warm text, cold blue focus, and the hard geometry of comic gutters. Its palette is derived from canonical references for the original Higgins 64 color palette used in the book, then mapped into a restrained desktop color system.

## Install

```bash
omarchy-theme-install https://github.com/OldJobobo/omarchy-watchmen-theme
```

Then select `watchmen` from the Omarchy theme picker, or run:

```bash
omarchy-theme-set watchmen
```

## What's Included

- Core Omarchy theme files for Hyprland, Hyprlock, Waybar, Walker, Mako, SwayOSD, GTK, icons, and shell colors.
- Terminal coverage for Foot, Ghostty, Kitty, Alacritty, and Warp.
- Extra app themes for Vencord, Zed, btop, Chromium, cliamp, and Neovim.
- A 17-wallpaper set in `backgrounds/`, with numbered release filenames.
- Palette references in `palette/`, including the reconstructed Higgins 64 source grid and the distilled UI palette.

## Palette

Watchmen uses `palette/watchmen64.yaml` as its color provenance: a reconstructed Higgins 64 grid based on the original graphic novel palette references. The active theme files then distill that source into Omarchy tokens, terminal colors, app overrides, and wallpaper treatments.

The result is not a generic comic-book palette. It leans into the specific tension of the printed pages: blue focus, violet-black shadow, muted steel, parchment text, worn reds, and controlled amber.
